    How It Works
    
        
    
          
    
            
    📡
    
            
    On-Chain Indexer
    
            
    Continuously scans and indexes the transaction graph of Sphere Nametags — analyzing frequency, volume, wallet age, and counterparty patterns.
    
          
    
          
    
            
    🛡️
    
            
    Sybil Detection
    
            
    Algorithmic scoring identifies multi-account spam structures. Low diversity + high frequency + new wallets = Sybil flag.
    
          
    
          
    
            
    🏆
    
            
    Credit Scoring
    
            
    Dynamic reputation index (0-100) with ratings from "Sybil" to "Excellent". Factors in AstridOS sandbox verification for bonus trust.
    
          
    
          
    
            
    💰
    
            
    Data Paywall
    
            
    Commercial agents pay 0.5 testnet tokens via P2P transfer to access scores. Verified payments unlock signed payloads via Nostr DMs.
    
          
    
          
    
            
    🤖
    
            
    Fully Autonomous
    
            
    Entire pipeline — scanning, scoring, billing, delivery — runs at machine speed with zero human intervention.
    
          
    
          
    
            
    🔒
    
            
    AstridOS Sandbox
    
            
    Agents verified to run inside AstridOS secure sandbox receive trust bonuses. Unverified environments are penalized.
